kwxmms2d

kwxmms2d ist ein simples Script mit welchem man den Server des X-Music-Multiplexing-System 2 konfigurieren. starten und stoppen kann.

Notiz: Dieses Script benutzt das Programm xmms2d.


1. Konfiguration

2. Server starten

3. Server stoppen


1. Konfiguration

Server: Eingabe der IP Adresse(n) auf dem der Server lauschen soll. Tragen Sie bitte die Adresse(n) der xmms2d Servers getrennt durch ein Semikolon ein (Bsp.: tcp://host:9667 oder unix:///tmp/xmms-ipc-username). Der IPC Socket "unix" ist nur für lokale clients vorgesehen, der Socket "tcp" funktioniert lokal wie remote.

Logdatei verwenden: Soll der XMMS2 Server seine Ausgaben in eine Logdatei schreiben (yes/no)?

Logdatei: Auswahl einer Logdatei. Standard ist $HOME/.cache/xmms2/xmms2d.log

Root: Soll der XMMS2 Server mit Root Rechten gestartet werden (yes/no)?


2. Server starten

Lokalen Server starten.


3. Server stoppen

Lokalen Server stoppen.
