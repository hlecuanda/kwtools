kwxmms2d

kwxmms2d ist ein simples Script mit welchem man den Server des X-Music-Multiplexing-System 2 konfigurieren. starten und stoppen kann.

Notiz: Dieses Script benutzt das Programm xmms2d.

1. Konfiguration

1.1. Server Konfiguration

Server: Eingabe der IP Adresse(n) auf dem der Server lauschen soll. Tragen Sie bitte die Adresse(n) der xmms2d Servers getrennt durch ein Semikolon ein (Bsp.: tcp://host:9667 oder unix:///tmp/xmms-ipc-username). Der IPC Socket "unix" ist nur für lokale clients vorgesehen, der Socket "tcp" funktioniert lokal wie remote.

Logdatei verwenden: Soll der XMMS2 Server seine Ausgaben in eine Logdatei schreiben (yes/no)?

Logdatei: Auswahl einer Logdatei. Standard ist $HOME/.cache/xmms2/xmms2d.log

Root: Soll der XMMS2 Server mit Root Rechten gestartet werden (yes/no)?

1.2. Plugin Konfiguration

output.plugin: Auswahl des Ausgabe Plugins das verwendet werden soll.

diskwrite.destination_directory: Auswahl eines temorären Verzeichnis in dem xmms2d Daten zwischen speichern kann.

alsa.device: Auswahl einer der von Alsa unterstützen Soundkarten.

cdda.device: Auswahl eines CD/DVD/Blueray Laufwerks.

curl.proxyaddress: Falls Sie einen Proxy in Ihrem Netzwerk nutzen können Sie ihn hier eintragen (Adresse:Port).

curl.useproxy: Soll ein Proxy benutzt werden (yes/no)?

curl.authproxy: Müssen Sie sich am Proxy authentifizieren (yes/no)?

curl.proxyuser: Eingabe des Benutzernamens für die Authentifizierung am Proxy.

curl.proxypass: Eingabe des Passworts für die Authentifizierung am Proxy.

mad.id3v1_encoding: Angabe des Zeichensatzes zum enkodieren von id3v1 Tags (z.B. ISO8859-1 oder UTF-8).

mpg123.id3v1_encoding: Angabe des Zeichensatz zu enkodieren von id3v1 Tags (z.B. ISO8859-1 oder UTF-8).

oss.device: Auswahl einer der von OSS unterstützten Soundkarten. Dies kann entweder eine zum Beispiel /dev/dsp0 Gerätedatei sein toder eine von ALSA nativen Gerätedateien mittels der Bibliothek libaoss, zum Beispiel /dev/snd/pcmC0DC0c (Paket alsa-oss).

oss.mixer: Auswahl der Mixer Gerätedatei /dev/mixer.
