kwgrub2

Installiert und konfiguriert den Bootloader grub2. Mit grub2 brauchen Sie nicht mehr den Bootloader neu zu installieren, falls Sie einen neuen Kernel kreiert haben. Genaueres gibt es mit dem Befehl "info grub". Man kann ihn in den Master Boot Record oder in den Bootsector einer Partition installieren.

Notiz: Benötigt root Rechte.

1 Konfiguration

Hier kann man die Konfigurationsdatei /etc/default/grub anlegen oder eine vorhandene verändern. Siehe dazu auch http://wiki.ubuntuusers.de/GRUB_2/Konfiguration.


2 Sicherheit

Hier können Sie einige Einstellungen für Passwortschutz von grub2 tätigen. Beachten Sie das diese Einstellungen in der Datei /etc/default/grub gespeichert werden.

Benutzername: Tragen Sie ein oder mehrere Benutzer getrennt durch ein Leerzeichen ein, die das Bootmenü bearbeiten dürfen.

Notiz: Folgende Passwort Einstellungen gelten immer für alle Benutzer.

Passwort: Geben Sie bitte ein Passwort für den/die Benutzer ein. Dieses ist ein Klartext Passwort und sollte nicht benutzt werden.

Passwort_pbkdf2: Hiermit können Sie ein verschlüsseltes Passwort erstellen. Dieses muss mindestens 8 Zeichen lang sein. Es wird daraus ein Hash erzeugt.


3 Aktualisierung

Mit diesem Menüpunkt können Sie die Bootmenü Datei (grub.cfg) von grub2 aktualisieren. Bei der Aktualisierung werden die Werte aus der Konfigurationsdatei /etc/default/grub benutzt.


4 Installation

Wohin soll grub2 installiert werden. In eine oder mehrere Festplatten oder in den Bootsector einer Partition. grub2 kann in verschiedene Festplatten installiert werden. Dies ist besonders bei RAID Arrays zu empfehlen. Ausserdem kann grub2 mit Festplatten groesser als 2,1 TerraByte umgehen.

Festplatte: Auswahl einer oder mehrere Festplatte(n).

Partition: Auswahl einer oder mehrere Partitionen.
