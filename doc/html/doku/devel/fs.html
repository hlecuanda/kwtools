Allgemeine Funktionen die das Dateisystem betreffen.

Man muss diese Funktion (fs_conf) nur ausführen, dann kann man alle enthaltenenen Funktionen ausführen, wie z. B. "copy_fs". Wenn nichts zu der Syntax steht, wird die Funktion einfach nur aufgerufen. Manche Funktionen haben auch noch eigene Funktionen die aber nur von diesen benutzt werden und dadurch bisher noch nicht weiter erklärt werden.

check_part_fs

Die Funktion check_part_fs überprüft das übergebene Laufwerk darauf welches Dateisystem es besitzt. Dabei werden die util-linux >=2.16.2 benutzt.
check_part_fs LAUFWERK
LAUFWERK - Variable die das zu überprüfende Laufwerk enthält.
Variable
gv_Fs - Variable die das Dateisystem enthält.

copy_fs

copy_fs kopiert Dateisysteme und passt sie der Grösse an. Zur Zeit werden nur die Dateisysteme ext2/3/4, fat16, fat32, ntfs, reiserfs, reiser4 und xfs unterstützt. Die Dateisysteme/Partitionstypen auf dem Quell- und Ziel-Laufwerk müssen identisch sein. fat16 und fat32 Dateisysteme kann man nicht vergrössern und xfs Dateisysteme nicht verkleinern.
copy_fs SRC DST
SRC - Variable die die Quell-Partition enthält.
DST - Variable die die Ziel-Partition enthält.

floppy_format

Diese Funktion dient zum formatieren von Disketten mittels "fdformat". Zum Beispiel um Images auf Diskette zu kopieren (Bootdiskette).
floppy_format "$FLOPPY"

format

format dient zum Formatieren von Blockorientierten Geräten, wie Festplatten. Zuerst wird nach den Programmen für die Dateisysteme gesucht. Wenn diese vorhanden sind, werden sie zur Auswahl angeboten, um die Partitionen, Logische Volumes oder RAID-Arrays dementsprechend zu formatieren.
format DEVICES [ FS ]
DEVICES - Variable die die zu formatierenden Blockgeräte enthält.
FS - Dateisystem, ist optional. Kann man mit übergeben, wenn die Funktion formatprogs_check schon verher ausgeführt wurde. Ansonsten ruft format diese Funktion auf.

formatprogs_check

Überprüft, welche Formatierungswerkzeuge installiert sind. Nur die aufgezählten gehen zur Zeit. Dabei wird die Variable FM_Menue belegt.
formatprogs_check [ -p | -s ]
-p = Überprüfung auf ext2, ext3, ext4, fat32, hfs, hfsplus, jfs, linux-swap, ntfs, reiserfs, reiser4, ufs, xfs. linux-swap wird nur von kwparted benötigt.
-s = Überprüfung auf swap.
nichts = Überprüfung auf ext2, ext3, ext4, fat32, hfs, hfsplus, jfs, ntfs, reiserfs, reiser4, ufs, xfs.
Variable
FM_Menue - Variable die die Namen der Dateisysteme enthält deren Werkzeuge installiert sind. Diese sind aufbereitet für die Funktion menubox.

mdadm_conf_scan

Diese Funktion dient zum updaten der Konfigurationsdatei des Programms mdadm (Software-RAID).
mdadm_conf_scan

resize_fs

Mit der Funktion resize_fs kann man die Dateisysteme von Partitionen, RAID Arrays, Logischen Volumes oder verschlüsselten Laufwerken ändern, vorrausgesetzt das Dateisystem unterstützt das.
resize_fs [ -l | -p | -r ] [+|-]SIZE OLD_SIZE DEVICE
-l - LVM in der Grösse verändern.
-p - Partitionen in der Grösse verändern.
-r - RAID Arrays in der Grösse verändern.
-+ - vergrössern.
-- - verkleinern.
SIZE - Grösse in KiloByte die dazu addiert oder entfernt werden soll.
OLD_SIZE - alte Grösse des Laufwerks, zur Berechnung.
DEVICE - Laufwerk, kann ein RAID, LV oder eine normale Partition sein.
Notiz: normale Partitionen (-p) gehen nocht nicht.

Dateisystemnachrichten und Fehlermeldungen
no_copy_fs_msg

Nachricht das man dieses Dateisystem nicht kopieren kann, da die Grösse des Ziellaufwerks zu klein ist.

no_fs_support_msg

Info darüber, wenn ein Dateisystem (noch) nicht unterstützt wird.
no_fs_support_msg FILESYSTEM
FILESYSTEM - Dateisystem welches noch nicht unterstützt wird.

no_fs_type_msg

Info darüber, wenn kein Dateisystemtyp festgestellt werden kann.
no_fs_type_msg DEVICE
DEVICE - Laufwerk dessen Dateisystemtyp nicht erkannt wurde.

no_space_fp_msg

Info darüber, dass kein Platz mehr auf dem Laufwerk ist. Lesen Sie dazu die Hilfe zum Scrip kwrsync_backup.
no_space_fp_msg [ -e DST_SIZE SRC_SIZE ]
-e - erweiterte Fehlermeldung mit Anzeige der Laufwerksgrössen.
DST_SIZE - Zielgrösse des Laufwerks.
SRC_SIZE - vorhandene Grösse des Laufwerks.
nichts - Fehlermeldung das der Platz nicht mehr ausreicht.

use_kwcryptsetup_msg

Info darüber das dieses Laufwerk verschlüsselt ist und mit dem Script kwcryptsetup bearbeitet werden muss.
